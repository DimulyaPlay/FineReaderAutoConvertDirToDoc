Программа для мониторинга папки и распознавания появляющегося в ней содержимого в Word с помощью API ABBYY FineReader, а также сохранения изображений страниц pdf на страницах Word.
Полезна для тех, у кого есть сетевые МФУ/сканеры, которые сканируют в заданную сетевую папку, а также для тех, кто прикрепляет сканы кассационных определений в базу ГАС Правосудие.

Может выполнять следующие задачи:
1. С помощью FineReader распознает файлы, появляющиеся в заданной папке и сохраняет указанную папку в формате word.
Возможно указание префикса, файлы с которым будут обрабатываться, остальные будут игнорироваться.
Форматы файлов не ограничены, в программу попадает все, поэтому, если что-то не поддерживается FineReader, ошибки в логе не будет, так как finecmd.exe не возвращает результат выполнения.

2. Преобразование сканов формата PDF, появляющихся в заданной папке в файлы word в виде изображений страниц в указанную папку.
Возможно указание префикса, файлы с которым будут обрабатываться, остальные будут игнорироваться.
Возможно задать DPI изображений для повышения качества, если требуется. По умолчанию 75 для экономии места.
Форматы файлов ограничены .pdf, остальное игнорируется.

Обе задачи могут выполняться как по отдельности, так и параллельно. Пути можно указывать для каждой задачи разные, или одинаковые. Ограничение: выходы не могут совпадать со входами.
Программа сканирует указанную директорию каждые N секунд, при обнаружении новых файлов запускает заданные процедуты, результаты с временными отметками записываются в главном окне программы

Установленный FineReader не должен иметь лицензионных ограничений на использование FineCmd.exe
Для запуска сканирования при запуске программы нужно открыть программу с параметром -start_scan
